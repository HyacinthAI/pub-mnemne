# pub-mnemne

Public materials for [Mnemne](https://mnemne.ai) — a private, local-first AI
knowledge platform from Hyacinth.

This repository accompanies the product. It is the place we publish documents,
methodologies, and disclosures that we want to be openly inspectable. Mnemne
itself is a commercial product and is not distributed from this repository.

## About Mnemne

> Your knowledge managed with local AI.

Mnemne keeps your data on your hardware. Summarization, and
question-answering run on the device — there are no remote servers in the
loop, no telemetry pipeline, and no cloud index of your content. The product
currently is offered in two forms:

- **Mnemne** — desktop app for macOS.
- **Mnemne Enterprise** — runs on an appliance inside the
  customer's data center for institutional knowledge management.
  
For product details, downloads, and pricing, see **<https://mnemne.ai>**.
